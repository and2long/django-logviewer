# django-yt-logviewer

A reusable Django app that adds a web interface for browsing project log files.

## Usage

1. Install the package:

   ```bash
   pip install django-yt-logviewer
   ```

2. Add the app to `INSTALLED_APPS`:

   ```python
   INSTALLED_APPS = [
       # ...
       "logviewer",
   ]
   ```
